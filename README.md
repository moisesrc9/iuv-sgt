# iuv-sgt
Sistema de Gestión del Transporte Público del Estado de Campeche

<b>Descripción del Proyecto:</b>
<br>
Diseñar y desarrollar el Sistema de Gestión del Transporte Público del Estado de Campeche (SGT) para la Agencia Reguladora del Transporte del Estado de Campeche (ARTEC), con el propósito de establecer una plataforma tecnológica que permita administrar la información de concesiones, vehículos, conductores y usuarios del sistema 

<b>Objetivo del Repositorio:</b>
<br>
Tener un control del versionamiento en el desarrollo del proyecto, se pretende desarrollar con PHP, JavaScript y con MySQL como motor de base de datos.

<b>Arquitectura y Metodología Aplicada:</b>
<br>
El Sistema de Gestión del Transporte (SGT) se desarrolla bajo una arquitectura en capas, la cual permite separar la presentación, la lógica de negocio y el acceso a datos. 
<br>
En la capa de presentación se implementa el patrón Modelo-Vista-Controlador (MVC), el cual permite organizar de manera estructurada la interacción entre la interfaz de usuario, la lógica del sistema y los datos.
<br>
El desarrollo se basa en el paradigma de Programación Orientada a Objetos (POO) y se emplea la metodología orientada a objetos.
